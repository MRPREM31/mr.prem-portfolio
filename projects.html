ClockLive: Productivity Dashboard

Published: August 25, 2025 Last updated: September 2, 2025

A beautiful, feature-rich dashboard for students and professionals with clock, timer, calculator, and weather features.

Key Features
  • Real-time clock with date and day display
  • Customizable timer with alarm sounds
  • Built-in calculator with calculation history
  • Real-time weather information based on location
  • Personalization options and multiple themes
  • Battery status display and screen wake lock
Technologies Used
  • HTML5, CSS3 with Glassmorphism design
  • Vanilla JavaScript for functionality
  • Geolocation API for weather data
  • Battery Status API for power monitoring
  • Screen Wake Lock API to prevent screen dimming
  • Local Storage for user preferences
